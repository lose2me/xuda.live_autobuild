## 宿舍是上床下桌吗？规格？<Badge text="床宽差异" type="warning" />
::: tabs#faq
@tab 一期
六人间上床下桌，独立卫浴、独立阳台。1-6号楼两间宿舍共用一个阳台；7号楼是新建楼栋，条件最好。床铺规格 0.9×2.0m，部分老床铺晃动时可能有噪音，可报修。

@tab 二期
六人间上床下桌，独立卫浴，配置与一期基本一致。床铺规格 0.9×2.0m。

@tab 东校区
六人间上床下桌，2023年竣工的新校区，设施较新，上床有楼梯式台阶。床铺规格 1.0×2.0m，比中心校区宽10cm，买床品注意尺寸。宿舍楼有电梯但平时不常开。

:::
## 教室和宿舍有没有空调？<Badge text="能耗等级" type="warning" />
::: tabs#faq
@tab 一期
大部分教室有空调（触屏操控，天热时通常会开），仅部分小教室没有。宿舍均有空调和暖气，空调为3级能耗，耗电较多，开学需缴400元租赁费。冬季暖气非常足，12月24小时不停供，室内穿短袖没问题。

@tab 二期
与一期基本一致，大部分教室有空调，宿舍有空调和暖气，开学需缴400元租赁费。

@tab 东校区
大部分教室有空调，部分面积小的专业教室仅有风扇。宿舍有空调和暖气，空调可自由开关，不断电。开学需交400元空调租赁费。

:::
## 有独立卫浴吗？<Badge text="有无差异" type="danger" />
::: tabs#faq
@tab 一期
有独立卫浴和独立阳台。7号楼有24小时热水，条件最好；其余楼栋使用太阳能热水器，冬天没太阳时热水不稳，需要去楼下公共澡堂。一期澡堂位于一食堂北边约50米，宿舍一楼配备免费吹风机。

@tab 二期
有独立卫浴，条件与一期基本一致。冬天太阳能不给力时同样需去公共澡堂。

@tab 东校区
无独立浴室。一二楼各有公共澡堂，隔板加浴帘，刷校园卡按次计费2元。中午12:30后有热水，夏天凌晨也有，但冬天晚洗可能没热水，尽量别拖太晚。浴室附近设有吹风机。

:::
## 洗澡热水供应时间？<Badge text="时间差异" type="warning" />
::: tabs#faq
@tab 一期
7号楼24小时热水。其余楼栋使用太阳能热水器，热水供应从早上5:00开始，冬天阴天不稳定，建议去公共澡堂。

@tab 二期
太阳能热水器供热水，热水供应从早上5:00开始，条件同一期非7号楼。

@tab 东校区
热水供应约13:00-22:00，上午虽可能有水但极少有人洗。冬天尽量别太晚去，避免没热水。

:::
## 有早自习、晚自习吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
早自习时间因专业而异，部分7:20到，部分7:50到（第一节课8:00开始），可咨询直系学长学姐。大一普遍有晚自习，上学期管理较严，下学期多无人参与。

:::
## 有晨跑吗？每学期有跑步打卡要求吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
没有晨跑，也无任何强制性跑步或打卡要求。

:::
## 寒暑假放多久？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
徐工以放假早闻名，考试安排早，考完即可离校。暑假通常两个月，寒假有时也达两个月。每学期按课表为19或20周。

:::
## 学校允许点外卖吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
允许。标注"校园送"的外卖及外部商家骑手均可送到宿舍楼下，也有学生运营的小程序可送至宿舍门口（费用较高）。需及时取餐。

:::
## 学校交通便利吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
离地铁站近，校门口有共享电动车（约2-3元可达地铁站）。仅通地铁2号线，半小时直达市中心彭城广场。学校不在市区，附近有小型商场满足日常需求。徐州东站（高铁站）迎新期间有接站大巴2元/人直达校内。

:::
## 食堂价格贵吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
与校外相近略低，多数菜品10-20元。全校四个食堂，每个食堂三层，每天提供免费汤。食堂内微信和支付宝都能用，但饮水机和浴室只能刷校园卡。高峰期12:00排队超20分钟，建议11:15左右到。有KFC、麦当劳、喜茶、瑞幸等连锁品牌入驻。

:::
## 宿舍楼有洗衣机吗？<Badge text="新旧差异" type="warning" />
::: tabs#faq
@tab 一期
一楼有洗衣房，约四十台滚筒洗衣机、两台烘干机、一台洗鞋机，通过软件或微信小程序使用。设备相对较旧。二楼或三楼有露台可晒大件。

@tab 二期
配置与一期类似，一楼洗衣房，通过软件或微信小程序使用。

@tab 东校区
每栋二楼有洗衣房，配洗衣机和洗鞋机。因校区较新且去年更换过，设备新且干净，通过软件或微信小程序使用。

:::
## 校园网怎么样？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
还不错。手机多使用流量，电脑多连校园网。一张校园卡的校园网最多连两个设备，覆盖范围广。建议自己办一个校园手机卡，不要借别人的账号。

:::
## 每天断电断网吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
宿舍24小时供电供网。但23:00至次日6:00熄灯限网（节假日和考试周除外）。

:::
## 校园内可以骑电瓶车吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
可以。开学初检查严格，电动车需装学校牌照（免费），风头过后检查较松。有专用充电区，可通过"铁塔充电"软件查询充电桩位置。校园较大，尤其跨校区上课的专业建议备一辆。

:::
## 宿舍限电情况？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
无明确限电规则，六人同时玩电脑没问题。但宿管判定违规电器不看功率，电吹风、煮蛋器、电煮锅甚至筋膜枪都可能被收走通报，打火机和酒也不能出现在宿舍。楼内有公共吹风机。

:::
## 通宵自习有去处吗？<Badge text="略微差异" type="warning" />
::: tabs#faq
@tab 一期
宿舍有通宵自习室，使用时需告知宿管，避免查房时被记夜不归宿。

@tab 二期
与一期类似，宿舍有通宵自习室，使用时需告知宿管。

@tab 东校区
自习室通常10-11点关门，不允许通宵。但宿舍楼一楼和二楼的自习空间无时间限制，可通宵使用。

:::
## 大一能带电脑吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
可以，宿舍内使用自由。需用电脑的课程可携带，无需的不建议带。电脑建议开学后再买，不用提前带。

:::
## 学校里面用什么卡？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
日常出行带手机即可，微信、支付宝消费。校园身份卡（开学免费发放，印有头像和学号）主要用于饮水机打水和浴室洗澡，这两项只能刷卡。校园卡丢了补办需去大学生活动中心，费用20元。

:::
## 学校会发银行卡吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
录取通知书里附一张建行卡，拿身份证去家附近任意建行网点就能激活，不用等开学排队。激活时把额度调到学费以上，学校会自动扣款。开卡工本费5元+账户管理费10元需提前预存。

:::
## 学校的超市怎么样？<Badge text="位置差异" type="warning" />
::: tabs#faq
@tab 一期
超市位于一食堂地下负一层，同层还有奶茶、理发店、移动电信营业厅、建行等生活配套。

@tab 二期
地下有"橙堡"商业区，罗森、茶百道、面包店、平价女装店、水果店、打印店一站式配齐。

@tab 东校区
三食堂一楼有一家面积较大的超市和麦当劳，三楼有零食店（比超市便宜但不能刷校园卡）和精品店、美甲店等商业区。

:::

## 门禁和查寝是什么情况？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
入校无需刷卡或刷脸。周日至周四22:00关门，周五周六延至22:30，夜间不归按夜不归宿处理。查寝频率高，基本天天查，时间约22:00-23:30不固定，在宿舍露个脸即可。错过了事后去宿管处签字说明，不会有处分。

:::
## 军训多长时间？需要准备什么？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
军训约两周，上午7:30-11:30，下午14:30-17:30。徐州夏天紫外线极强，必备：SPF50+防晒霜、硅胶鞋垫（军训鞋底极薄）、1.5L以上大水杯、腰带（军训裤不配）。冰袖不让穿，降温靠休息时的小风扇。身体不适立刻报告，请假需辅导员签字+医院证明。

:::
## 转专业难吗？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
大一上、大一下各一次机会。大一上笔试不考转入专业课，相对简单；大一下增加专业课考试，难度加大。成绩需在本专业前30%-40%，热门专业（计算机、会计）要求绩点3.0+，挂科直接失去资格。只能在高考专业组内转，不能文理跨组。

:::
## 挂科有什么后果？<Badge text="无差异" type="note" />
::: tabs
@tab 无差异
期末挂科后下学期有一次免费补考，但补考通过也算挂科记录，奖学金、评优、入党资格全部取消。补考未过需重修，重修费1学分100元。挂科达3门学校会联系家长。通识课挂科不算正式挂科记录，不影响评奖评优。

:::
## 学长&学姐评价<Badge text="无差异" type="note" />
::: tabs#faq
@tab 一期
学长：虽然我才上了一年，不过我觉得都挺好，都能接受，我爱徐工。除非你是那种公子哥，大小姐啥的。

@tab 二期
与一期评价类似，整体环境和条件差不多，一期二期隔天桥相连，生活配套共享。

@tab 东校区
学姐：非常真诚啊，我不属于那种抱怨这抱怨那的，我觉得都挺好，都能接受，我爱徐工。

:::
